WritePad Editor

Steps to open WritePad editor:
  1. Click Start
  2. Click Programs
  3. Click WritePad

Advantages of using a WritePad editor:

  • It allows faster cretion of Web pages.
  • It allows easier updation of text or images in WritePad editor.
  • It is platform independent.

Disadvantages of using a WritePad editor:
  • It consumes more memory space.
  • It repuires to fill-in more details such as meta data for each page.
  • It is platform independent.
